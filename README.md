<h1 align='center'>Natours</h1>
<p align='center'>An awesome tour booking site built on top of NodeJS.</p> </br>
<h2>Key Features</h2>
<ul>
  <li>Authentication and Authorization</li>
  <ul>
    <li>Login and logout</li>
  </ul>
  <li>Tour</li>
  <ul>
    <li>View all Tours, check user's reviews and rating </li>
  </ul>
  <li>User Profile</li>
  <ul>
    <li>Update username, email, and password </li>
  </ul>
</ul>

<h2>Build With</h2>
<ul>
  <li>Express - The web framework used/li>
  <li>Mongoose - Object Data Modelling (ODM) library</li>
  <li>MongoDB Atlas - Cloud database service</li>
  <li>Pug - High performance template engine</li>
  <li>JSON Web Token - Security token</li>
  <li>Postman - API testing</li>
</ul>
